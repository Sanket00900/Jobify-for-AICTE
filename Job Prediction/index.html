Highlights of our Forcasting System

Taking Inputs

We will focus on mainly 2 inputs from the user which are course he wants to opt for and keywords he focuses primarily to center his job. So we will be getting the user more focused on his/her skills and ambition.

Web Scraping

We will be extracting content and data that the user focuses on and prioritizes from the websites as mentioned above and this information will be saved in the database for eventually defining the report.

Loading of data

Here we will be scraping the data obtained for every ‘X’ time and loading it into the database. Over which an analysis will be performed and these analysis attributes shall be stored into a new database. After which we will free the main loading database for the next ‘X’ iteration.

Machine Learning Model

The scrapped data will use the information, which are the inputs of tweets for sentimental analysis, for the past 1 month for training a machine learning model for attaining better accuracy. The trained model then will be used for fetching the skills of the required jobs. On a similar basis, training the model will be done by using the past 24hrs on the testing data.

Report

After the following steps are done, a report will be generated highlighting details such as the number of jobs, that jobs trend depending on its views from past and its scope in the future, public opinions through sentiment analysis, and also EDA and some other analysis that contribute the been factor.
